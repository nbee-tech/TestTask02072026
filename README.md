# Тестовое Задание

Проект на C#, в котором реализованы несколько сервисных классов и автоматические тесты для проверки их поведения.

## Структура проекта

Основная логика находится в namespace `Services`.

Внутри него расположены сервисы:

* `LogService` — обработка и нормализация логов из входного файла.
* `CompressorService` — сжатие строки по повторяющимся символам.
* `ServerService` — работа со счётчиком с учётом параллельного доступа.

Для каждого сервисного класса добавлены автоматические тесты, которые проверяют ожидаемый результат работы методов.

## Архитектурный подход

Проект разделён на основную логику и тесты:

* `TestTask02072026` — основной проект с сервисами.
* `TestTask02072026.Tests` — проект с unit-тестами.

Я намеренно применял подходы и конструкции, которые совместимы с legacy-кодом: обычные классы, традиционные namespace-блоки и явные сервисные методы.

## Тестирование

Тесты написаны с использованием xUnit.

Запуск тестов:

```bash
dotnet test
```
